# Oriviu UI

## _UI Component for Oriviu WebApps_

Oriviu is a SaaS in the education sector that connects students, teachers and parents.

## Installation

Oriviu requires Node.js v14+ to run.

```sh
git clone https://github.com/wisnuvb/oriviu-ui.git
cd oriviu-ui
npm i
npm start
```
